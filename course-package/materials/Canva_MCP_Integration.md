# Canva MCP 串接與簡報生成操作說明

> 對應單元：**u-3 用 AI 生成課程簡報（雙路徑實作）— 路徑一**｜對象：學員
>
> 本指南教你如何在 Claude 中一鍵授權 Canva，讓 Claude 直接在你的 Canva 帳號後台建立一份可編輯、可匯出 PPTX 的精美簡報。全程瀏覽器操作。

---

## 一、什麼是 MCP？（一句話版）

**MCP (Model Context Protocol)** 是讓 Claude 這顆「大腦」接上外部工具「手腳」的標準介面。開啟 Canva 連接後，Claude 就能直接幫你在 Canva 裡排版投影片，而不只是給你文字。

> 路徑一（Canva MCP）適合「需要精美圖文、之後還要進 Canva 微調、要匯出 PPTX」的情境。若你要的是「快速、可離線、純網頁」的簡報，請改用 [Claude_Artifacts_Guide.md](Claude_Artifacts_Guide.md)（路徑二）。

---

## 二、開啟 Canva 連接授權（一次設定，日後免重做）

1. 先確認你已能登入 [Canva](https://www.canva.com)。
2. 回到 [Claude.ai](https://claude.ai)，點擊左下角**你的頭像 / 名稱 → 設定 (Settings)**。
3. 找到 **連接器 / 擴充功能 (Connectors / Extensions / MCP)** 分頁。
4. 在清單中找到 **Canva**，點擊 **連接 (Connect)**。
5. 瀏覽器會跳出 Canva 授權視窗，登入並點擊 **允許 (Allow)**。
6. 回到 Claude，看到 Canva 顯示「已連接」即完成。

> 若清單中沒有 Canva，請確認你的工作區是學校的 **Team 版**，且系統管理員已開放連接器功能。

---

## 三、用一句話生成簡報

在對話框輸入下列 Prompt（**粗體字換成你的課程主題**）：

> 請在 **Canva** 中幫我建立一份 **8 頁**的簡報，主題是**「2026 年流行色彩趨勢與服裝設計實務」**，請套用**現代簡約風格**的設計模板，每頁加上標題、3 個重點條列，並為主色調加上一段配色意境描述。

Claude 會調用 Canva MCP，在你的 Canva 後台建立投影片，並回傳一個連結。

---

## 四、進 Canva 微調與匯出 PPTX

1. 點擊 Claude 回傳的 **Canva 連結**，會直接開啟該份設計。
2. 在 Canva 中可以：更換圖片、調整字體、拖動版面、換配色。
3. 匯出：右上角 **分享 (Share) → 下載 (Download) → 檔案類型選 `PowerPoint (.pptx)`** → 下載。
4. 下載後的 `.pptx` 可在教室的 PowerPoint 直接開啟播放。

---

## 五、好用的追加指令（在同一對話接著說）

- 「第 3 頁的配色改成以**雲舞暖白**為底、**變革深藍綠**為標題色。」
- 「幫我在最後加一頁『課堂討論題』，列 3 個問題。」
- 「把每頁的文字精簡成不超過 4 行，適合上課投影。」

---

## 六、常見問題

- **Q：Claude 說找不到 Canva？** 請回到步驟二重新確認授權狀態，並重新整理頁面。
- **Q：生成的圖片不合適？** 進 Canva 後手動替換即可，Canva 內建大量素材庫。
- **Q：一定要連網嗎？** 是。路徑一需要連線；若教室網路不穩，建議改用路徑二 Artifacts。
