# Claude Artifacts 互動簡報生成與離線下載指南

> 對應單元：**u-3 用 AI 生成課程簡報（雙路徑實作）— 路徑二**｜對象：學員
>
> 本指南教你用 Claude Artifacts 快速產生一個「可互動、可離線播放」的 HTML 網頁簡報。全程瀏覽器操作，不需要 Canva 帳號、不需要連網播放。

---

## 一、Artifacts 是什麼？

**Artifacts** 是 Claude 的即時預覽面板。當你請 Claude 產生網頁、簡報或小工具時，它會在對話右側開一個獨立視窗，讓你**當場看到成品並互動**。適合：

- 想要**快速**做出一份投影片，不追求精緻圖片。
- 教室**網路不穩**，希望能離線全螢幕播放。
- 想要**可點擊、可翻頁**的互動效果。

> 若你需要精美圖庫與可匯出 PPTX，請改用路徑一 [Canva_MCP_Integration.md](Canva_MCP_Integration.md)。

---

## 二、一句話生成互動簡報

在 [Claude.ai](https://claude.ai) 對話框輸入（**粗體字換成你的主題**）：

> 幫我針對**流行設計系**的**「2026 流行色彩趨勢」**主題，使用 **Artifacts** 生成一個**極簡、有設計感、可互動、可用左右鍵翻頁**的 HTML 課程網頁簡報，適合上課投影使用。內容包含：封面、4 個主色介紹（每個色附色票與意境文字）、一頁課堂討論題、一頁結語。

Claude 會在右側 Artifacts 面板打開一個可翻頁的簡報。

---

## 三、下載成 HTML，離線播放

1. 在 Artifacts 面板右上或右下角，找到 **下載 (Download)** 或 **發佈 (Publish)** 按鈕。
2. 選擇下載為 **HTML 檔**，存到桌面（例如 `color-trends-2026.html`）。
3. 上課時，**用瀏覽器直接開啟這個檔案**（雙擊即可），按 F11 或瀏覽器全螢幕即可投影。
4. 這份檔案是自包含的，**不需要網路**也能播放與翻頁。

---

## 四、常用的追加調整（在同一對話接著說）

- 「配色改用**變革深藍綠 (#004f54)** 當主色、**雲舞暖白 (#f4f1ea)** 當背景。」
- 「字放大一點，投影時後排也看得清楚。」
- 「幫每一頁加上頁碼，並在封面加上我的姓名與課程名稱。」
- 「加一個進度條顯示目前第幾頁。」

## 五、路徑一 vs 路徑二 快速對照

| 比較項目 | 路徑一：Canva MCP | 路徑二：Claude Artifacts |
|---|---|---|
| 圖片素材 | 豐富（Canva 圖庫） | 無內建圖片（以色塊、排版為主） |
| 匯出格式 | PPTX（進 PowerPoint） | HTML（瀏覽器播放） |
| 是否需連網播放 | 是 | 否，可離線 |
| 互動效果 | 一般簡報 | 可翻頁、可互動 |
| 適合情境 | 要精美、要再編輯 | 要快速、要離線 |

---

## 六、常見問題

- **Q：下載的 HTML 打不開？** 確認是用瀏覽器（Chrome / Edge）開啟，而不是用文字編輯器。
- **Q：可以事後再改嗎？** 回到原本的 Claude 對話，繼續下指令請它修改，再重新下載即可。
- **Q：可以轉成 PPTX 嗎？** Artifacts 產出的是 HTML；若最終一定要 PPTX，建議改走路徑一。
