## 上傳方式（講師操作）

1. 登入 Dify.ai 雲端團隊版 → 「知識庫」→ 建立知識庫，命名為 `設計學院-院級共用庫`。
2. 將本資料夾三份 `.md` 一次拖入，分段設定選「自動分段並清洗」。
3. 等狀態變「可用」後，用「召回測試」輸入上表的檢索情境句驗證。
