# Dify AI Agent 與 Claude Desktop 視覺影音備課實戰工作坊 — 課程總覽

## 基本資訊
- **對象**: 設計時尚學院教師（視覺傳達、數位媒體、創意產品、流行設計、服飾設計、時尚經營等科系，共 10 位，無資訊背景）
- **總時數**: 4 小時 (1 天 × 4 小時)
- **上課時間**: 2026 年 7 月下午半天
- **教室**: 設計學院專業電腦教室
- **講師**: AI 輔助教學設計專家團隊

## 課程目標
1. 能夠在 Windows (PowerShell) 環境中獨立建置 Claude Desktop、Git、Node.js、Python 等 AI 本地端協作環境。
2. 能夠熟練操作 `npx skills add` 指令一鍵安裝並使用 `superpowers`、`taste-skill` 等外掛技能包。
3. 能夠呼叫 `extract-design-system` 分析知名網頁設計 Tokens，並讓 AI 擔任視覺設計作品的評審助教，提供符合 WCAG 對比度與視覺層級的修改回饋。
4. 能夠藉由 AI 編寫 `p5.js` 互動幾何配色網頁，並使用 `hyperframes` 進行影音程式渲染。
5. 能夠利用本地 Claude Desktop 呼叫雲端 Dify RAG API 讀取課綱與設計教材，實現雲地一體化備課流程。

## 共用案例
**創藝時尚設計工作室的 AI 升級轉型案**  
我們將虛構一家「創藝時尚設計工作室 (Creative Fashion Studio)」，以該工作室需要進行「品牌重塑、2026 色彩行銷、網頁設計系統重構與社群影片宣傳」為核心任務。所有單元實作（簡報生成、網頁設計系統分析、作品評審、互動配色網頁、宣傳短片生成以及備課 API 串聯）都將圍繞此案例展開，使老師們能以完整的實戰情境理解 AI Agent 如何在創意設計領域落地應用。
