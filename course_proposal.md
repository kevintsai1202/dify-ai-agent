# 課程規劃案：「Dify AI Agent 與 Claude Desktop 視覺影音備課實戰工作坊」

本課程專為**設計時尚學院**教師（視覺傳達、數位媒體、創意產品、流行設計、服飾設計、時尚經營等科系）量體打造。考慮到教師多數無資訊背景，課程設計採用最直覺的**「Skills (技能) 先行、漸進學習」**架構。

為了確保現場實作的順暢度，所有技能安裝均改為「直接安裝整包技能，並加上 `-y` 參數跳過互動式提示」，避免 Windows 底下交互詢問出錯。

同時，本課程特別針對**設計系教師的教學與改作業情境**，設計了**「AI 視覺設計助手與評分顧問」**技能示範。老師們不僅能體驗 AI 寫程式與影片的威力，還能學會利用 AI 提取網頁設計系統、自動審查視覺色彩對比度，並讓 AI 擔任「改學生設計作業的助手」，給予極具設計專業的留白與排版優化建議。

---

## 📌 課程基本資訊

*   **對象**：設計時尚學院教師（共 10 位，無資訊背景）
*   **時長**：4 小時（半天工作坊）
*   **核心工具**：Dify (團隊版)、Claude Desktop (桌面版)、HyperFrames
*   **教學目標**：
    1.  完成 AI 本地端開發環境（Claude Desktop, Git, Node.js, Python）建置。
    2.  學會安裝 **Superpowers、Taste-Skill 與 PPTX 等實體技能包**。
    3.  利用 **視覺輔助技能 (如設計系統提取)** 進行案例教學，並讓 AI 提供設計排版建議與對比度審查。
    4.  體驗 **Visual & Video Coding**：讓已具備設計超能力的 Claude 幫忙生成精美的網頁互動教材（`p5.js`）與渲染學生作品影片（**HyperFrames**）。
    5.  體驗 **雙向整合**：由本地端 Claude Desktop 呼叫雲端 Dify RAG API，打通雲端與本地備課工作流。

## ⏱️ 4 小時課程大綱

### 🕒 第一小時：AI 本地環境建置與 Dify 雲端入門
*   **主題**：*「無痛起步：踏入 AI 程式與 Agent 的世界」*
*   **授課模式**：講師演示 + 教師跟隨操作
*   **時間分配**：
    *   **00:00 - 00:40 (40分鐘)：手把手環境建置實戰**
        *   引導全體教師在 Windows (PowerShell) 電腦上安裝：
            1.  **Claude Desktop** 官方視窗版。
            2.  **Git**（用於版本管理與 CLI 工具支援）。
            3.  **Node.js (LTS)**（執行 JavaScript 與網頁工具的基礎環境）。
            4.  **Python 3.10+**（注意勾選「Add Python to PATH」）。
    *   **00:40 - 01:00 (20分鐘)：Dify 平台基本介紹與 RAG 觀念**
        *   講師演示 Dify 雲端後台介面，說明什麼是 AI Agent、Workflow 以及 RAG 智慧檢索的運作原理。

---

### 🕒 第二小時：Claude Desktop 的 Skills 安裝與日常備課應用
*   **主題**：*「不寫程式，一鍵幫桌面端 AI 裝上設計與簡報整包超能力」*
*   **授課模式**：講師案例示範 + 教師同步體驗
*   **時間分配**：
    *   **01:00 - 01:30 (30分鐘)：三大核心技能 (Skills) 的一鍵免提示安裝**
        *   **觀念**：了解什麼是 **Agent Skills**（AI 的功能擴充包），就像手機安裝 App 一樣簡單。
        *   **功能說明**：介紹我們將安裝的三個分開的 Skill，以及它們對設計教學的強大助力：
            1.  `obra/superpowers` — 🧠 **思維超能力**：引導 AI 養成嚴謹規劃、先思考後動手的工作習慣。
            2.  `leonxlnx/taste-skill` — 🎨 **設計師品味**：內置高階視覺設計與配色排版規範，使 AI 產出內容具備高階審美。
            3.  `anthropics/skills` — 📄 **官方基礎技能**：外掛微軟 PowerPoint 簡報直接產出（pptx 技能）與檔案讀寫能力。
        *   **同步安裝實作**：引導老師在終端機分別輸入以下指令（加上 `-y` 參數跳過詢問直接安裝）：
            *   `npx skills add obra/superpowers -y`
            *   `npx skills add leonxlnx/taste-skill -y`
            *   `npx skills add anthropics/skills -y`
    *   **01:30 - 02:00 (30分鐘)：內建搜尋 + 簡報技能日常備課實作**
        *   **講師示範**：使用內建聯網搜尋與剛裝好的 `anthropics/skills` 中的簡報功能，老師用中文說：`「幫我上網搜尋 2026 年最新流行色彩趨勢，並直接在我的桌面生成一份 8 頁、套用 taste-skill 設計美感的服裝設計備課 PowerPoint 簡報（包含大綱與配色意境描述）。」`
        *   **跟隨體驗**：老師在自己電腦輸入範例對話，體驗簡報在桌面瞬間生成的驚喜效果。

---

### 🕒 第三小時：視覺設計輔助技能 (AI 評審) 與 視覺影音程式生成
*   **主題**：*「讓 AI 當你改學生作業的助手，並體驗影片渲染與互動配色網頁」*
*   **授課模式**：講師操作示範 + 教師跟隨體驗
*   **時間分配**：
    *   **02:00 - 02:25 (25分鐘)：視覺設計輔助技能（設計系統提取與改作業助手）**
        *   **安裝提取技能**：使用指令 `npx skills add arvindrk/extract-design-system -y`，安裝設計系統提取技能。
        *   **講師演示 1：提取案例網站設計系統**
            *   輸入任一優良設計網站網址，讓 Claude 自動分析並提取其「色彩 Tokens (色碼)、字體搭配與間距比例」，作為課堂教學案例。
        *   **講師演示 2：AI 擔任改設計作業助教**
            *   講師上傳一張「學生設計網頁/海報的截圖」，並利用已安裝的 `taste-skill` 請 Claude 進行評審。
            *   **成果**：Claude 像專業設計總監一樣，給出極具美學素養的「文字對比度安全檢索 (Accessibility Audit)、負空間 (Negative Space) 留白建議與排版視覺流向 (Visual Hierarchy) 修改建議」，對老師評分與給予學生回饋大有幫助。
    *   **02:25 - 02:40 (15分鐘)：Visual Coding (視覺編程) 實作**
        *   **同步操作**：命令已具備高階設計審美的 Claude Desktop 寫出 `p5.js` 程式，生成互動幾何配色網頁，雙擊點開即可在瀏覽器內互動體驗。
    *   **02:40 - 03:00 (20分鐘)：HyperFrames Video Coding (影音編程) 實作**
        *   **安裝影片技能**：使用指令 `npx skills add heygen-com/hyperframes -y` 一鍵安裝影片生成技能。
        *   **影音渲染**：命令 Claude 幫忙寫 HTML 影片結構，自動渲染成學生作品短影片 (MP4)。老師跟著做簡單的影片生成測試。

---

### 🕒 第四小時：Claude Desktop 呼叫 Dify API (RAG Skill) 與 Q&A 問題診斷
*   **主題**：*「雲地串聯整合示範、雙導師備課實戰與互動問答」*
*   **授課模式**：講師核心示範 + 教師同步測試 + 互動交流
*   **時間分配**：
    *   **03:00 - 03:30 (30分鐘)：利用 Claude Desktop 呼叫 Dify API (自製本地 Dify Skill)**
        *   **講師示範**：老師在雲端 Dify RAG 上傳教材 PDF 並啟用 API 取得金鑰。接著在本地安裝 Dify 官方串接工具整包：`npx skills add langgenius/dify -y`。將雲端金鑰配置完成。
        *   **工作流演示**：講師在 Claude Desktop 中說：`「幫我讀取剛才在 Dify 建置的服飾設計大綱，針對第一週內容寫一份 300 字講義與小測驗。」`，Claude 自動調用 Dify 技能讀取雲端課綱並在本地快速生成 HTML 教學內容。老師同步在本地跟著完成測試。
    *   **03:30 - 04:00 (30分鐘)：🎤 Q&A 互動交流、自由提問與教學應用診斷**
        *   **捨棄發表壓力**，留出整整 30 分鐘，由講師與老師們針對「日常備課、教案設計、數位創作教學如何結合這些 AI 工具組合」進行自由提問與深度交流，提供各科系老師客製化的 AI 備課落地建議。

## 🛠️ 課前準備與教材提示（學校承辦人與教師準備清單）

為了讓第一小時的「環境安裝」進行得更順利，建議課前完成：

1.  **下載連結懶人包**：
    *   課前準備好一個網頁或講義，提供 **Claude Desktop, Git, Node.js, Python (Windows Installer)** 的直接下載連結，避免老師現場搜尋下載到錯誤的版本。
2.  **API 帳號預先註冊**：
    *   請老師於課前先註冊好 **GitHub 帳號**與 **Dify.ai 團隊版帳號**，以省下現場註冊與收驗證信的時間。
3.  **教學素材與截圖準備**：
    *   請每位老師攜帶 **3-5 份自己的教學大綱 PDF**。
    *   攜帶 **1-2 張學生過往的設計作品截圖 (網頁、海報或衣服設計)**，用於第三小時 AI 評審顧問的互動測試。
